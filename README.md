# editor-de-texto-c
Editor de texto simples feito em C. Possui as funções de criar, salvar, editar e excluir arquivo.
Utilizou-se alocação dinâmica de memória (malloc()) e listas ligadas para fazer um arquivo de texto com um número de linhas desejável pelo usuário. Feito durante o 2° semestre de Eng. de Computação, enquanto curso Estrutura de Dados.
Dificuldades na criação:
1 - Meu primeiro projeto real. Foi difícil saber como estruturar cada função e imprimir as informações no terminal.
2 - Não tinha conhecimento sobre funções do tipo fopen, fprintf e nem conhecia o tipo de variável FILE.
3 - Saber limpar o terminal de maneira efetiva.
No geral, foi difícil construir esse programa com os conhcimentos obtidos nos 2 primeiros semestres, mas aprendi bastante.
