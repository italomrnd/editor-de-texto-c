# editor-de-texto-c
Editor de texto simples feito em C. Possui as funções de criar novo arquivo, abrir arquivo e salvar arquivo.
Utilizou-se alocação dinâmica de memória (malloc()) e listas ligadas para fazer um arquivo de texto com um número de linhas desejável pelo usuário. Feito durante o 2° semestre de Eng. de Computação, enquanto curso Estrutura de Dados.
